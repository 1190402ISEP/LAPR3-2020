- Foi nos pedido que considerassemos uma bateria de 15000mAh para o drone, mas como na nossa aplicação
	não utilizamos essa unidade de medida, convertemos ess valor para Watts hora assumindo que o drone
	tem uma voltagem de 17.6V (usamos como referência drone DJI Mavic 2 PRO). 
- Para a conversão usamos o seguinte site: https://www.rapidtables.com/calc/electric/mah-to-wh-calculator.html
- Obtivemos 264 Wh de bateria seguindo esta lógica.
